Les tests en Javascript

Le but de cette présentation est de montrer que mettre en place des tests dans une application n'est pas insurmontable quand nous utilisons les bons outils

En tant que développeur, je suis fan du framework Angular. Mais les outils que j'utilise, Jasmine, Karma ou Protractor peuvent être utilisés quelque soit le contexte. Certes Protractor n'est pas conseillé par la team Google pour les applications non Angular mais c'est un formidable wrapper pour accéder aux web drivers. Cette application sert de support à la présentation permettant de présenter ces outils.
